# WMS业务功能&单据
## 问题点
1. 空的容器是否需要进行库位管理? 
    - 管理，空容器进行移动时，需要进行转移操作，操作不便。
    - **不管理，拆托时进行判断是否为空，为空则去掉与库位之间得联系。** 

## 收货
![avatar](png/arrival.png)
- 业务需求
    - 货拉到仓库门口，仓库进行收货，见流程图
          
       1. 无来源收货，直接扫序列或者批次，
       2. 有来源收货

- 业务单据
    - 批次 数量   序列   库位   容器(可为空)   任务明细id
